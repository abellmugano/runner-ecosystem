## Descrição
Phase 7: SDK scaffolding. Estrutura para criar módulos/SDKs para o Runner Ecosystem.

## Estrutura criada
- `runner-system/sdk/commands/create.ps1`, `build.ps1`, `publish.ps1`.
- `runner-system/sdk/config/sdk-config.psd1`.
- `runner-system/sdk/templates/template-default.ps1`.
- `runner-system/sdk/tests/sdk.tests.ps1`.

## Como testar
1. Mudar para a branch phase7-sdk (criada a partir de main).
2. Executar os comandos de scaffold para criar módulos e simular build/publish.
